
# lucky-fn  
lucky-fn是一个可配置可定制化的抽奖应用，炫酷3D球体，可用于年会抽奖等活动，支持奖品、人员、界面、图片音乐配置。

> 如果进入网站遇到图片无法显示或有报错的情况，请先到【全局配置】-【界面配置】菜单中点击【重置所有数据】按钮清除数据后进行更新。

> 该项目将在近期进行**内部代码重构**及**开发新功能**，预计元旦节前三天上线新版本。
